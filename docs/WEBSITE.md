# Website Architecture

The site is a *visualization layer* generated by Quarto from the repository — it never stores
canonical content. Planned/standard features:

- Version selector (per release/tag) · Language selector (en · es-AR · pt-BR)
- Full-text search · Citation export (CITATION.cff + per-page) · Downloads
- Glossary (auto-generated from concept frontmatter) · Cross references (auto-resolved)
- Roadmap · News · Issues/Discussions links · Dark mode (reader mode)

Hosting: GitHub Pages, custom domain `research.schadler.tech`. Linked from the institutional site
`https://schadler.tech` (see project handoff notes). Design: minimal, institutional, timeless
(OECD/ISO/MIT/W3C register) — no flashy effects.
