# Naming Conventions

- Folders & files: `kebab-case`. Markdown for content, `.qmd` for documents needing computation.
- Concept ids: `concept-NNNN` (stable, never reused). RFCs: `NNNN-slug.md`. Papers: `WP-NNNN-slug`.
- Versions: SemVer (`MAJOR.MINOR.PATCH`) per object and for the Framework as a whole.
- Stable identifiers are forever: never rename an official concept id; supersede instead.
