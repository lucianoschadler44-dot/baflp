# ontology

> Part of **BAFLP** — The Buenos Aires Framework for Artificial Legal Personhood.
> This README is mandatory metadata. Do not delete.

## Purpose
Formal vocabulary of entities and relations (what exists in the Framework and how things relate).

## Contents
Versioned ontology files (`ontology.vX.Y.md`), entity definitions, relation types, an `ONTOLOGY.md` index.

## Dependencies
Authority: Chief Research Architect. Referenced by every concept via `relations:` metadata.

## Future evolution
Migrates toward a machine-readable graph (e.g. RDF/Turtle export) once stable.
